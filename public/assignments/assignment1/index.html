This site provides an extremely brief introduction to Operating Systems and some of the main concepts behind them.


An Operating System, such as Windows or MacOS, is tool that allows for a computing device to be simpiler to use and program.
The main component of the Operating System is the Kernel.
The Kernel is the piece of software that interfaces directly with the hardware components.
Some of the most important components of the Kernel are those of process management, memory management, and file systems.
Due to the complexity of computers these days (there is a multidtude of different components that go inside a computer and even when the same models are being compared,
there is no guarentee the have the exact same configuration)it has become necessary for there to be a unified system that developers can target otherwise each developer
would have to know the exact configuration and would not be able to have slightly portible code.
Another benefit of this is modern day Operating Systems also have the user in mind and bundle different Desktop Enviroments that allow users to navigate and make full
use of the system which increases use-ability for the general public.
